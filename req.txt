ระบบล็อคอิน แยก user และ admin 1
Admin และ ผู้จัดการ หน้ารายงาน จะมีปุ่ม อนุมัติ และไม่อนุมัติ 0
และสถานะ การจองห้อง เช่น รอตรวจสอบ, อนุมัติ, ไม่อนุมัติ สามารถใส่เหตุผลได้ 0


หน้ารายงาน
- โชว์สถานะการจอง
- User จะเห็นแค่รายงานของตัวเอง และมีแค่ปุ่ม แก้ไข
- Admin จะเห็นของทุกคน และมีปุ่ม ✔(อนุมัติ) และ X (ไม่อนุมัติ) และ ดวงตา

- ปุ่มดวงตา จะเด้ง Modal และแสดงรายละเอียดการจองทั้งหมด และสามารถเปิด PDF เพื่อนำไปให้หัวหน้าเซ็น
